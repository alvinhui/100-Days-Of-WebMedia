# 在 HTML 表单中进行媒体捕获

1. 启动本地 HTTP 服务：

  ```bash
  $ npm install http-server -g
  $ http-server
  ```
2. 通过 Chrome 访问有 IP 地址的本地 HTTP 服务
3. 在 Chrome 页面上点击鼠标右键，生成当前网页的二维码
4. 通过手机默认浏览器扫码访问查看演示效果
